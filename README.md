# Electroquimica MATLAB

Repositorio de scripts MATLAB para practicas, tareas y rutinas de electroquimica.

## Catalogo

| Carpeta | Script | Tema |
| --- | --- | --- |
| `CV/` | `rutina_CV_Au_micro_macro.m` | Rutina de voltametria ciclica para comparar Au microelectrodo vs Au macroelectrodo. |
| `EIS/` | `Nyquist_EIS_Biosensor_Study.m` | Simulacion y analisis de diagramas Nyquist para biosensor. |
| `EIS/` | `Nyquist_EIS_Biosensor_Study2.m` | Variante del estudio Nyquist/EIS. |
| `EIS/` | `DOE_Hu_MOF_EIS_Simulator.m` | Simulador DOE para EIS en MOF. |
| `Polarizacion/` | `tarea2_polarizacion_mixta.m` | Polarizacion mixta. |
| `Tafel/` | `Tafeltutorial.m` | Tutorial y analisis Tafel. |

## Nota de uso

La rutina `CV/rutina_CV_Au_micro_macro.m` busca los archivos `.txt` exportados por CHI760D en una carpeta `080526` ubicada junto al script y guarda resultados en `resultados_CV`.
